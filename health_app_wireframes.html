Notas de Design UX - Adaptado para MIT App Inventor

Principais Funcionalidades (Simplificadas):

  • Onboarding e Autenticação: Sistema de login local simples com TinyDB
  • Dashboard: Cards informativos básicos com ações rápidas
  • Monitoramento de Sinais Vitais: Entrada manual completa com armazenamento local
  • Gerenciamento de Medicamentos: Lembretes básicos e controle de aderência
  • Registro de Sintomas: Escala de dor simples e registro de texto
  • Relatórios de Saúde: Resumos textuais e cálculos básicos
  • Consultas: Lembretes simples de consultas agendadas
  • Perfil e Configurações: Dados básicos e configurações locais

Limitações e Adaptações do MIT App Inventor:

  • ✅ Funcionalidades Viáveis: Entrada manual de dados, armazenamento local, interface básica
  • ❌ Limitações Aceitas: Sem integrações complexas, sem sincronização em nuvem, apenas Android
  • 🔄 Abordagem Simplificada: Foco total em entrada manual e armazenamento local (TinyDB)
  • 💡 Solução Educacional: Prioriza aprendizado e funcionalidade básica sobre recursos avançados
  • 📱 Componentes Básicos: TextBox, Button, ListView, TinyDB, Notifier para funcionalidade completa

Princípios de UX Adaptados para App Inventor:

  • Simplicidade: Interface limpa usando componentes básicos do App Inventor
  • Funcionalidade: Foco em recursos essenciais que realmente funcionam
  • Navegação Simples: Telas organizadas com botões claros de navegação
  • Feedback Básico: Notificações e alertas usando componente Notifier
  • Armazenamento Local: Dados seguros no dispositivo com TinyDB
  • Usabilidade: Interface intuitiva para usuários de todos os níveis técnicos
  • Educacional: Demonstra conceitos de desenvolvimento mobile de forma prática

Componentes do App Inventor Utilizados:

  • Layout: VerticalArrangement, HorizontalArrangement para organização
  • Entrada: TextBox, Button, CheckBox, Spinner para interação
  • Exibição: Label, ListView, Image para mostrar informações
  • Armazenamento: TinyDB para dados locais persistentes
  • Funcionalidades: Clock, Notifier, DatePicker para recursos básicos
  • Navegação: Screen para diferentes telas do aplicativo

Objetivos Educacionais Alcançados:

  • Desenvolvimento Mobile: Experiência completa com plataforma visual
  • Gestão de Dados: Compreensão de armazenamento local e persistência
  • Interface de Usuário: Aplicação prática de princípios de UX/UI
  • Limitações Técnicas: Adaptação criativa dentro de restrições de plataforma
  • Projeto Completo: Do wireframe à implementação funcional
